Matrix Power Series

Time Limit: 3000MS
Memory Limit: 131072K
Total Submissions: 17928
Accepted: 7581

Description
Given a n ¡Á n matrix A and a positive integer k, find the 
sum S = A + A2 + A3 + ... + Ak.

Input
The input contains exactly one test case. The first line of input contains 
three positive integers n (n ¡Ü 30), k (k ¡Ü 109) and m (m < 104). Then 
follow n lines each containing n nonnegative integers below 32,768, 
giving A's elements in row-major order.

Output
Output the elements of S modulo m in the same way as A is given.

Sample Input
2 2 4
0 1
1 1

Sample Output
1 2
2 3
